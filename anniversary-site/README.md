# Anniversary Site

A premium, romantic, cinematic React application created as part of the Surprise Websites project.

## Features
- Deep charcoal romantic theme with warm glowing accents
- Reusable timeline for story milestones
- Premium photo gallery with modal view
- Animated envelope revealing a love letter
- "Reasons I Love You" interactive grid

## Getting Started

First, install the required dependencies:
```bash
npm install
```

Then start the development server:
```bash
npm run dev
```

## Adding Your Content
To make this site your own, replace the data and images found in the following locations:
- `src/data/timelineData.js`
- `src/data/galleryData.js`
- `src/data/reasonsData.js`
- `src/components/letter/LoveLetterSection.jsx`
- Replace images in `public/images/`
- Add your own background music to `public/music/anniversary-theme.mp3`

## Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- Lucide React (Icons)
