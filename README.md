# Surprise Websites

A premium, romantic multi-page React website project containing two separate applications: `anniversary-site` and `birthday-site`.

## Getting Started

1. Navigate to either `anniversary-site` or `birthday-site`.
2. Run `npm install` to install all dependencies.
3. Run `npm run dev` to start the local development server.

## Folder Structure

- `anniversary-site/`: A romantic, cinematic, and emotional application designed for anniversaries.
- `birthday-site/`: A joyful, premium, celebratory application designed for birthdays.
- `shared-assets/`: Holds global placeholders like photos, videos, notes, and songs to be used across the sites.

## Adding Content
Place your actual images, videos, music, and text inside the respective `public` or `src/assets` folders in each app, or use the `shared-assets` folder as reference for what needs to be included.
